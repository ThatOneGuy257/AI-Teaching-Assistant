# AI-Teaching-Assistant
An automated teaching assistant built with Python that helps track class progress, generate lesson briefs, and oganize teaching materials.

## Current Features
- Connects to Google Sheets
- Retrieves Class information
- Tracks class progress and slide numbers

## Planned Features
- Genereate daily teaching summaries with AI
- Create lesson preparation notes
- Track assingments and quizzes
- Write daily briefs to Google Docs
- Automate reminders before classes

## Technologies Used
- Python
- Google Sheets API
- Google Docs API
- AI Language Models
- Git/Github
